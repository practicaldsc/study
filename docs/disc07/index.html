Problem 9

Consider a dataset of n values, y_1, y_2, ..., y_n, all of which are non-negative. We‚Äôre interested in fitting a constant model, H(x) = h, to the data, using the new ‚ÄúWolverine‚Äù loss function:

L_\text{wolverine}(y_i, h) = w_i \left( y_i^2 - h^2 \right)^2

Here, w_i corresponds to the ‚Äúweight‚Äù assigned to the data point y_i, the idea being that different data points can be weighted differently when finding the optimal constant prediction, h^*.

For example, for the dataset y_1 = 1, y_2 = 5, y_3 = 2, we will end up with different values of h^* when we use the weights w_1 = w_2 = w_3 = 1 and when we use weights w_1 = 8, w_2 = 4, w_3 = 3.


Problem 9.1

Problem 9.2
